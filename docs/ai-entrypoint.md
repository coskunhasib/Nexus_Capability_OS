## How to classify fields

Each field should be classified as one of:

```text
canonical — likely source-of-truth field
adapter-derived — generated by an adapter or worker
runtime-derived — generated by runtime event flow
ui-only — used for display/export only
temporary — fixture/mock/trial-only
unknown — cannot be trusted yet
```
